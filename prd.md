# **하루 시간표 시각화 웹 애플리케이션**

## **1. 개요**

본 문서는 사용자가 자신의 하루를 계획하고 시각적으로 확인할 수 있는 인터랙티브 웹 애플리케이션의 요구사항을 정의합니다. 사용자는 원형 시계 위에서 시간을 드래그하여 활동을 추가하고, 각 활동에 이름, 아이콘, 색상을 지정할 수 있습니다. 추가된 활동은 시계와 하단 목록에 표시되며, 언제든지 수정하거나 삭제할 수 있습니다.

## **2. 주요 기능 및 구현 현황**

### **STEP 1: 기본 프로젝트 구조 설정 (완료)**
- **Vite + React** 기반의 프로젝트를 생성하여 빠른 개발 환경을 구축했습니다.
- `src` 폴더 내에 컴포넌트 기반의 모듈식 구조를 설계했습니다.
  - `components`: 재사용 가능한 UI 컴포넌트 (시계, 모달, 활동 목록 등)
  - `hooks`: 비즈니스 로직 및 상태 관리 커스텀 훅
  - `assets`: 폰트, 이미지 등 정적 자산
- `App.css`에 기본 스타일과 Google Fonts (Noto Sans KR)를 적용하여 일관된 디자인을 유지합니다.

### **STEP 2: 원형 시계 UI 구현 (완료)**
- **ClockWheel.jsx**: SVG를 사용하여 24시간을 나타내는 동적인 원형 시계 컴포넌트를 구현했습니다.
- **ClockRing.jsx**: 시계 눈금(시간 표시)을 생성하여 가독성을 높였습니다.
- **Sector.jsx**: 사용자가 추가한 활동을 시계 위에 부채꼴 모양으로 렌더링합니다. 각 섹터는 활동의 시작과 끝 시간을 시각적으로 나타냅니다.
- **DragHandler.jsx**: 사용자가 마우스로 시간을 드래그할 때 선택된 시간 범위를 시각적으로 표시하는 컴포넌트입니다.

### **STEP 3: 드래그로 시간 선택 기능 구현 (완료)**
- **useDragAngle.js**: 시계 중심점을 기준으로 사용자의 마우스 드래그 각도를 계산하는 커스텀 훅을 구현했습니다. 이를 통해 사용자는 직관적으로 시간 범위를 선택할 수 있습니다.
- `onMouseDown`, `onMouseMove`, `onMouseUp` 이벤트를 사용하여 드래그 시작, 진행, 종료를 감지하고 상태를 업데이트합니다.

### **STEP 4: 할 일 입력 모달 구현 (완료)**
- **ActivityModal.jsx**: 사용자가 시간 선택을 완료하면, 활동의 세부 정보를 입력할 수 있는 모달 창이 나타납니다.
- 초기 버전에서는 활동 이름만 입력받아 저장하는 기능을 구현했습니다.

### **STEP 5: 아이콘 및 색상 선택 기능 추가 (완료)**
- `ActivityModal.jsx`를 확장하여 사용자가 활동에 대한 **아이콘**과 **색상**을 선택할 수 있는 기능을 추가했습니다.
- `react-icons` 라이브러리를 활용하여 다양한 아이콘을 제공합니다.
- 미리 정의된 색상 팔레트에서 원하는 색상을 선택하여 활동을 시각적으로 구분할 수 있습니다.
- 선택된 아이콘은 `Sector.jsx`에 표시되어 시계 위에서 활동을 직관적으로 식별할 수 있게 합니다.

### **STEP 6: 활동 목록 및 편집/삭제 기능 구현 (완료)**
- **ActivityList.jsx**: 시계 하단에 사용자가 추가한 모든 활동의 목록을 표시하는 컴포넌트를 구현했습니다.
- 각 목록 항목에는 활동의 아이콘, 이름, 시간, 그리고 **수정** 및 **삭제** 버튼이 포함됩니다.
- **수정**: '수정' 버튼을 클릭하면 해당 활동의 정보가 미리 채워진 `ActivityModal`이 열려 내용을 쉽게 변경할 수 있습니다.
- **삭제**: '삭제' 버튼을 클릭하면 확인 절차 없이 해당 활동이 즉시 삭제됩니다.

## **4. 향후 개선 과제**

- **데이터 영속성**: 현재는 새로고침 시 모든 데이터가 사라집니다. `localStorage`나 Firebase 같은 데이터베이스를 연동하여 데이터가 유지되도록 개선할 수 있습니다.
- **반응형 디자인**: 모바일 등 다양한 화면 크기에서도 최적의 사용성을 제공하도록 반응형 디자인을 적용합니다.
- **드래그 앤 드롭으로 순서 변경**: 활동 목록에서 드래그 앤 드롭으로 활동 순서를 변경하는 기능을 추가할 수 있습니다.
- **테마 기능**: 사용자가 직접 애플리케이션의 테마(라이트/다크 모드)를 변경할 수 있는 기능을 제공합니다.
